# ERAFTS – Emergency Response Analytics & Failure Tracking System

## Overview

ERAFTS is a full-stack emergency incident management platform designed to support emergency monitoring, incident tracking, analytics, and reporting. The system enables administrators and users to manage emergency incidents efficiently through a modern dashboard interface.

---

## Features

### Incident Management

* Create emergency incidents
* Update incident status
* Delete incidents (Admin only)
* Search incidents

### Analytics Dashboard

* Total incident count
* High-risk incident tracking
* Medium-risk incident tracking
* Low-risk incident tracking

### Role-Based Access Control (RBAC)

* Admin Login
* User Login
* Admin-only operations
* User-restricted access

### Reporting

* Professional PDF report generation
* Incident summary statistics
* Incident details export

### Emergency Categories

* Medical Emergencies
* Fire Incidents
* Flood Incidents
* Accidents

### Mapping

* Live incident location visualization
* Interactive map integration

## Installation

### Clone Repository

```bash
git clone https://github.com/Pavithraa2005/ERAFTS-Emergency-Response-System.git
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
node server.js
```

### Open Application

```text
http://localhost:5500/frontend/login.html
```
